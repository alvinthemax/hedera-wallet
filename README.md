# Hedera Mainnet Wallet

Wallet sederhana untuk mengirim HBAR dan mengecek balance di jaringan Hedera Mainnet.

## Fitur

- ✅ Cek balance dengan private key
- ✅ Kirim HBAR ke alamat lain
- ✅ Tambahkan memo ke transaksi
- ✅ Client-side only (private key tidak dikirim ke server)
- ✅ Support Mainnet dan Testnet
